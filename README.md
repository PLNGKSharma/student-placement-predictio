# student-placement-prediction system

This project predicts student placement changes using Machine Learning

##Technologies Used
.Python
.Pandas
.Scikit-learn
.Streamlit

##Features
.Predicts placement chances
.Uses CGPA,internships,projects,coding skills,communication skills,and aptitude scores
.Interactive Streamlit web app
.Machine learning model with 80%+ accuracy

##How to Run

1.Install required libraries
pip install pandas scikit-learn streamlit

2.Run the Streamlit app
streamlit rum app.py
